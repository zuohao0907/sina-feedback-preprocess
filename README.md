# 新浪网用户研究用户反馈数据预处理
## 要求
### 端内文件
* 表头和示例文件一致（一般只需注意“ID”和“具体问题”栏）
### 端外文件
* 文件名应为文件来源，具体为：iOS, 华为, 小米, VIVO, OPPO, 魅族, 微博(注意iOS后两个字母大写)

## 功能
### 端内文件
1. 合并文件
2. 按时间排序
3. 去除重复（did、问题类型、具体问题均相同则去除）
4. 去除活动数据（自定义活动类别）
5. 分类名修改/合并
6. 检查无效分类并修改
7. 输出文件

### 端外文件
1. 合并文件
2. 按时间排序
3. 指定分类
4. 输出文件

## 步骤
1. 选择时间
2. 上传文件
3. 调整数据
4. 下载文件
